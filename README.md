# Admin Dashboard Portal

A full-stack **Admin Dashboard** built with **React** (frontend) and **Node.js/Express** (backend) with **JWT authentication**. This portal allows admins to:

- View their profile and stats
- Approve or reject students
- Add, update, or delete courses
- Manage and view course lists
- Logout securely

---

## 🛠 Features

### Admin:
- View admin profile and total counts
- View, approve, or reject student registrations
- Manage courses:
  - Add new courses
  - Edit existing courses
  - Delete courses
- Simple reporting dashboard (placeholder)

### Technology Stack:
- **Frontend:** React, Axios, CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB (via Mongoose)
- **Authentication:** JWT
- **Dev Tools:** dotenv, CORS, Nodemon
